# CP3406 Educational App

Android educational app project for JCU CP3406 Assessment 3, built with Kotlin and Jetpack Compose.

## Project status

The repository currently contains the initial Android Studio project scaffold. Features will be developed and committed incrementally so the repository records continuous project progress.

## Assessment scope

The completed app will provide at least four core screens:

- Landing page for key information, shortcuts, and navigation
- Activity screen for the main learning interaction
- Settings screen for user preferences
- User statistics screen for progress and learning metrics

The project is also expected to demonstrate:

- Jetpack Compose and Material Design 3
- Navigation with modern Android APIs
- ViewModels, repository pattern, and dependency injection
- Internet API integration
- Room persistence for statistics and relevant data
- Model unit tests and Compose UI tests
- Accessible, inclusive, age-appropriate, and privacy-conscious design

## Technology

- Kotlin
- Jetpack Compose
- Material Design 3
- Gradle Kotlin DSL
- Minimum Android SDK 24

## Getting started

1. Clone the repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync and install the required Android SDK components.
4. Run the `app` configuration on an emulator or Android device.

`local.properties`, signing credentials, environment files, and generated build artifacts are intentionally excluded from version control.

## Development workflow

Work will be split into focused commits for project architecture, individual screens, data sources, testing, accessibility, and documentation. The README will be updated as implementation decisions and user-facing features are finalized.
